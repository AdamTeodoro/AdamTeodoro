Entendendo conceitos iniciais

O que é programação?

Programar é escrever um código que representa tarefas que o computador vai executar.

O que é linguagem de programação?

A linguagem de programação é um conjunto de regras, sintáticas e semânticas, capazes de informar instruções ao computador.

Ferramenta para escrita de código

Primeiro você deve fazer o download de um editor de código ou IDE de sua preferência, a IDE ou o editor vão te auxiliar durante o processo de escrita do código. Embora você pode editar o código até mesmo no bloco de notas, o editor vai aumentar seu desempenho e permitir que você altere seu código de maneira personalizada.

Estou recomendando fazer o download do atom ou do vscode.
